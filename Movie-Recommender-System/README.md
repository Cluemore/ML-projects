# Movie Recommender System

A machine learning-based movie recommendation system built using Python, Pandas, Scikit-learn, and Streamlit.

## Features

- Movie recommendation using cosine similarity
- Clean Streamlit UI
- Responsive layout
- Automatic download of similarity matrix using Google Drive

## Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit
- Pickle
